## Key Differentiators Highlighted

The landing page clearly communicates Narrata's unique value vs. generic AI tools:

1. **Truth-based content library** – All content grounded in actual work history
2. **Job-to-profile matching** – Instant fit analysis (Role, Company, Strengths)
3. **Streaming insights** – Real-time feedback during draft generation
4. **Human-in-the-loop** – User maintains full control, AI suggests
5. **Reusable library** – Every paragraph saved and tagged for future use
6. **PM Levels built in** – Automatic level assessment and competency mapping

---

## Tone & Positioning

**Voice**: Clear, confident, product-forward  
**Audience**: Mid-career Product Managers actively applying to jobs  
**Differentiation**: Strategic storytelling system, not generic AI generation

**What We Avoid**:
- Generic AI hype ("powered by LLMs", "revolutionary")
- Competitor name-dropping (differentiate through features)
- Technical jargon (embeddings, RAG, etc.)
- Invented features not in product

**What We Emphasize**:
- Speed + quality (not either/or)
- Truth fidelity (zero hallucination)
- Transparency (see your fit in real time)
- Control (approve every word)

---

## Single CTA Strategy

**Primary CTA**: "Get Started (Free)"  
**Placement**: Hero section + Bottom of page  
**Supporting copy**: "Takes 3 minutes. No credit card required."

No pricing tiers, no multiple CTAs, no navigation confusion. Every element drives toward one action: start using the product.

## How to Use This Deliverable

### For Design Phase
1. Use text wireframe as layout structure
2. Follow visual hierarchy exactly as specified
3. Implement responsive breakpoints (mobile, tablet, desktop)
4. Match tone: clean, modern, SaaS-standard aesthetic

### For Development Phase
1. Implement single-page structure (no subpages)
2. Ensure CTA tracking (analytics on button clicks)
3. Optimize images (WebP format, lazy loading)
4. Test on all devices and browsers

### For QA Phase
1. Verify all 11 screenshots are placed correctly
2. Test CTA flow (click → onboarding)
3. Validate SEO meta tags
4. Check mobile responsiveness
