## ūüĒß Root Cause Analysis

### Why Industry & Business Model Succeed (97%+)

‚úÖ **Explicitly stated** in most JDs  
‚úÖ **Clear prompt instructions** with examples  
‚úÖ **Strong normalization logic** (e.g., "B2B SaaS")  
‚úÖ **Company descriptions** typically include this info

### Why Vertical & Segments Fail (2-3%)

‚ĚĆ **Less explicit** in JDs (often implied, not stated)  
‚ĚĆ **Prompt allows null too easily** ("return null if not mentioned")  
‚ĚĆ **LLM is conservative** - defaults to `null` when uncertain  
‚ĚĆ **Insufficient inference examples** in prompt

## ūüí° Recommended Fix

### Current Prompt (Too Permissive)
```
BUYER SEGMENT:
- Identify the primary buyer segment(s)
- Look for: "enterprise", "mid-market", "SMB"
- Return null if not mentioned or implied ‚ĚĆ <-- Too easy to return null
```

### Proposed Prompt (Encourages Inference)
```
BUYER SEGMENT:
- STRONGLY ENCOURAGED: Infer from context ‚úÖ
- Look for explicit mentions: "enterprise", "mid-market", "SMB"
- INFER from clues:
  * "Enterprise platform" ‚Üí "Enterprise"
  * "Small business software" ‚Üí "SMB"
  * "Consumer app" ‚Üí "Consumer"
- Return null ONLY if absolutely no clues exist ‚úÖ
```
